masmx.7r3A [Build 3] binaries

Changes from 7r3 to 7r3A Build 3
masmx.7r3A revises and makes more consistent the assembly
of hierarchically structured and named code and data. Earlier
application code needs no changes

added -H option, allows unrounded floating constants

addition of :z suffix on number constants for 7-word constant
size-suffixes may be used to position floating exponent field in label-equated values
in addition to overriding self-sizing in integers

addition of freestanding utility symbol
extracts exported assembly labels
and makes them available as environmental variables
