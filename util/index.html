These utilities are

        a number of simple utilities on osx.x86 osx.ppc ubuntu.x86 Windows32


                usea	another grep, for Windows plain command prompt and Unix shells
			You can pipe lists of files to usea



				dir /B /S [PATH] |  usea -z[a][i] string [string...]

				-z	no filenames follow after option -z
				-a	display only lines with all the search strings in them
				-i	search case-transparent
				-f	display names of files containing search strings
				-nf	display names only of files with no search strings

				usea file [file...] -z string [string...]


                seeif	another diff, for Windows plain command prompt and Unix shells


				paths are either two files or two directories

				seeif [-avVWRs] path path

			        -a      text, ignore stuff like CR
			        -v      display differences
			        -V      keep comparing after the first difference
			                try to resync text files

			        -W      treat whitespace fields as one space
			                ignore blank lines

			        -R      recurse subdirectories
			        -s      say if files are equal



                data	another DOS line editor, for Windows plain command prompt and Unix shells
			text-command driven, but Windows stdin allows inline editing



				data [-n][i] file

				-n	show line numbers
				-i	open edit in input mode. Unconditionally replaces the file

			for safest use and especially for scripts and gang update

				data from_file to_file [ < commands_file ] [ -options ]

				-g	colours (not MS command prompt)

			commands during data operation

				q		exit and don't write

				.		exit and write

				n		line number display on / off

				l [mask]	locate next instance mask or earlier mask on lines from current line

				lc [mask]	locate all instances on lines from current line

				L [mask]	locate next instance mask or earlier mask case-transparent on lines from current line

				LC [mask]	locate all instances case-transparent on lines from current line

				p[lines]	display lines from current line

				p!		display all lines from current line

				o[lines]	display lines from next line

				o!		display all lines from next line

				return		(command mode) advance one line

				return		(input mode) change to command mode

				-		move to previous line

				+[lines] [save_file]	advance lines and optionally file scanned lines

				+:line [save_file]	advance to line and optionally file scanned lines

				ireturn		with no space change to input mode

				i [text]	enter a line with optional text

				b [text]	enter a line with optional text before the current line

				r [text]	replace current line

				c °[from]°[to]°[-]		change all from to to in current line
							° is any character
							void from: c°°to° adds to at front of line
		 					c°°to°- adds to at back of line

				t °[from]°[to]°[lines][-]	translate all from to to for lines or for all lines from current position
							° is any character
							void from: c°°to° adds to at front of lines
							c°°to°- adds to at back of lines

				C		c hit case-transparent

				T		t hit case-transparent 

				d		remove current line

				d-		remove current line and go to previous line

				line,line [save_file] remove from line to line  and optionally file removed lines

				a file		add text from file at current position

			data acts as a viewer until a change is made. temp0.txt and temp1.txt contain updated versions up to exit




                fds	dump format file viewer for Windows plain command prompt and Unix shells



				fds file [-Ze]

				-Z	display all the file without waiting for keyins

				-e	does not add LF to lines streamed with -Z option

			dumps data in 16-octet lines, CR for next line

			a new display offset may be entered, decimal or leading zero for hex
			a search string can be entered with the new offset and may have embedded space 	

				fds ~/dynamic.rta/pduimage
				0000000000000000: 00000c00001200000100001400000300   "................"
				0000000000000010: 000000000100001c0000030000000000   "................"05000 green beans
				000000000000547a: 677265656e206265616e730000000000   "green beans....."


                csum    generates and displays expected IP / TCP / UDP checksums.
                        Input is quartets of dump-format hex symbols


				csum [ < file_of_4symbol_hex_words ]

				csum
				4500
				003c
				.
				.


		flo	displays IEEE754 64-bit floating values in hex notation


				
				flo
				1.9
				.
				.


                stripcr	writes stdin to stdout discarding CR

				stripcr < withCR > without

		stuffcr writes stdin to stdout insaerting CR before LF

				stuffcr < withoutCR > with
